---
title: "Town command reference (/town, /t)"
description: "Complete reference for /town and its alias /t — create towns, claim land, manage residents, set taxes and permissions, handle jails, and configure diplomacy."
---

The town commands are the primary tool for creating and managing player-owned towns. Every town action — from founding a settlement and claiming new land, to setting taxes and jailing outlaws — is available through `/town` or its alias `/t`.

<Note>
Most subcommands require you to be a resident of a town, and many require the Mayor or Assistant rank. Permission nodes follow the pattern `towny.command.town.<subcommand>`.
</Note>

## Creating and joining towns

<Warning>
Creating a town costs money (configured by the admin). Once deleted, a town and all its claim data are permanently removed.
</Warning>

```bash
# Found a new town named Riverdale
/town new Riverdale

# Join an open town
/town join Riverdale

# Invite a player to join your town
/town invite Steve

# Accept an invite to join a nation
/town invite accept

# Leave your current town
/town leave
```

---

## Claiming land

<Note>
The number of chunks you can claim is determined by the town level, bought bonus blocks, and server configuration.
</Note>

```bash
# Claim the chunk you are standing in
/town claim

# Claim the adjacent unfilled area automatically
/town claim auto

# Claim a 3-chunk circle around you
/town claim circle 3

# Claim a 2-chunk rectangle around you
/town claim rect 2

# Fill-claim all contiguous reachable chunks
/town claim fill

# Claim a new outpost at your location
/town claim outpost

# Unclaim the current chunk
/town unclaim

# Unclaim all town land (destructive)
/town unclaim all
```

## Town bank

```bash
# Deposit 500 into the town bank
/town deposit 500

# Withdraw 200 from the town bank (mayor only)
/town withdraw 200

# View the last 10 transactions
/town bankhistory

# Purchase 10 extra town blocks
/town buy bonus 10
```

## Ranks and residents

<AccordionGroup>
  <Accordion title="Built-in ranks">
    Towny ships with a default set of ranks defined in `townyperms.yml`. Common default ranks include **Assistant**, **Sheriff**, and **Builder**. Admins can add custom ranks in that file.
  </Accordion>
</AccordionGroup>

```bash
# Grant Steve the Assistant rank
/town rank add Steve Assistant

# Remove the rank
/town rank remove Steve Assistant

# See all ranked residents
/town ranklist

# List all town members
/town reslist

# List online members only
/town online
```

## Town settings (`/town set`)

```bash
# Set the message-of-the-day board
/town set board Welcome to Riverdale!

# Move the homeblock to your current chunk
/town set homeblock

# Move the spawn point to your feet
/town set spawn

# Rename the town
/town set name Springfield

# Set a 4-character town tag
/town set tag SPFD

# Set flat daily resident tax
/town set taxes 10

# Set the per-plot daily tax
/town set plottax 2

# Set default plot sale price
/town set plotprice 50
```

## Town toggles

```bash
# Allow explosions in the town
/town toggle explosion on

# Disable fire spread
/town toggle fire off

# Enable PvP across the whole town
/town toggle pvp on

# Make the town open (anyone can join)
/town toggle open

# Make the town spawn publicly accessible
/town toggle public
```

## Outlaws and jail

```bash
# Outlaw a player
/town outlaw add Griefer123

# Remove them from the outlaw list
/town outlaw remove Griefer123

# Jail a resident in cell 1 for 2 days
/town jail Steve 1 2

# Release a jailed resident
/town unjail Steve

# List jail cells
/town jail list
```

---

## Outposts

Outposts are non-contiguous claims placed anywhere in the world.

```bash
# Teleport to outpost 1
/town outpost 1

# List all outposts
/town outpost list

# Set the spawn location of outpost 1 (stand at new spawn first)
/town set outpost 1
```

---

## Buying and selling towns

<Warning>
Selling a town transfers full ownership to the buyer. This action cannot be undone.
</Warning>

```bash
# Put the town up for sale at 5000
/town forsale 5000

# Remove from sale
/town notforsale

# Buy a town that is listed for sale
/town buytown Riverdale
```

## Listing and discovery

```bash
# List all towns, sorted by residents (default)
/town list

# Sort by balance
/town list balance

# Sort by number of town blocks
/town list townblocks

# Other sort options: name, online, open, public, founded, ruined, bankrupt, upkeep, forsale

# Show info about the town at your location
/town here

# Find towns within 500 blocks
/town nearby 500
```
