---
title: "TownyAdmin commands (/townyadmin)"
description: "Complete reference for /townyadmin and its alias /ta — admin tools for managing towns, nations, residents, economy, toggles, backups, and server maintenance."
---

The `/townyadmin` command (alias `/ta`) gives server operators full administrative control over Towny. With it you can create, delete, and modify towns and nations without being a member, force economy operations, toggle server-wide settings, purge stale data, and trigger backups or database reloads.

<Warning>
Most `/townyadmin` commands are irreversible. Actions such as deleting a town, purging residents, or resetting banks permanently alter data. Always back up before running destructive commands in production.
</Warning>

<Note>
The root permission node is `towny.admin`, which grants all admin capabilities. Individual subcommand nodes follow the pattern `towny.command.townyadmin.<subcommand>`.
</Note>

## Subcommand overview

| Command | Description | Permission node |
|---|---|---|
| `/ta backup` | Create an immediate data backup. | `towny.command.townyadmin.backup` |
| `/ta checkperm <player> <node>` | Check whether a player has a permission node. | `towny.command.townyadmin.checkperm` |
| `/ta checkoutposts` | Verify and repair outpost claim data. | `towny.command.townyadmin.checkoutposts` |
| `/ta givebonus <town> <amount>` | Give a town bonus town blocks. | `towny.command.townyadmin.givebonus` |
| `/ta newday` | Manually trigger the new-day event. | `towny.command.townyadmin.newday` |
| `/ta newhour` | Manually trigger the new-hour event. | `towny.command.townyadmin.newhour` |
| `/ta purge <days>` | Delete residents inactive for more than N days. | `towny.command.townyadmin.purge` |
| `/ta reload` | Reload all Towny configuration files. | `towny.command.townyadmin.reload` |
| `/ta reset` | Reset Towny data (dangerous). | `towny.command.townyadmin.reset` |
| `/ta tpplot` | Teleport to the plot at given coordinates. | `towny.command.townyadmin.tpplot` |
| `/ta unclaim` | Forcibly unclaim the chunk you are standing in. | `towny.command.townyadmin.unclaim` |
| `/ta mysqldump` | Dump the database to a SQL file. | `towny.command.townyadmin.mysqldump` |
| `/ta database save\|load\|remove` | Save, load, or remove database entries. | `towny.command.townyadmin.database` |
| `/ta townyperms` | Reload the townyperms.yml file. | `towny.command.townyadmin.townyperms` |
| `/ta install` | Run the initial Towny setup wizard. | `towny.command.townyadmin.install` |

## Toggle (`/ta toggle`)

| Command | Description | Permission node |
|---|---|---|
| `/ta toggle debug` | Toggle debug logging. | `towny.command.townyadmin.toggle.debug` |
| `/ta toggle devmode` | Toggle developer mode output. | `towny.command.townyadmin.toggle.devmode` |
| `/ta toggle npc` | Toggle NPC resident creation. | `towny.command.townyadmin.toggle.npc` |
| `/ta toggle wildernessuse` | Toggle wilderness item-use protection. | `towny.command.townyadmin.toggle.wildernessuse` |
| `/ta toggle regenerations` | Toggle plot regeneration tasks. | `towny.command.townyadmin.toggle.regenerations` |
| `/ta toggle townwithdraw` | Toggle whether mayors can withdraw from the bank. | `towny.command.townyadmin.toggle.townwithdraw` |
| `/ta toggle nationwithdraw` | Toggle whether kings can withdraw from the bank. | `towny.command.townyadmin.toggle.nationwithdraw` |

```bash
# Enable debug logging
/ta toggle debug

# Disable town bank withdrawals server-wide
/ta toggle townwithdraw
```

| `/ta set capital <nation> <town>` | Change a nation's capital town. | `towny.command.townyadmin.set.capital` |

| `/ta town rank <town> <add\|remove> <player> <rank>` | Modify town ranks. | `towny.command.townyadmin.town.rank` |
| `/ta town set <town> <key> <value>` | Set a property on a town. | `towny.command.townyadmin.town.set` |
| `/ta town toggle <town> <flag>` | Toggle a flag on a town. | `towny.command.townyadmin.town.toggle` |
| `/ta town outlaw <town> <add\|remove> <player>` | Manage a town's outlaw list. | `towny.command.townyadmin.town.outlaw` |
| `/ta town deposit <town> <amount>` | Deposit money into a town bank. | `towny.command.townyadmin.town.deposit` |

| `/ta town bankhistory <town>` | View a town's bank history. | `towny.command.townyadmin.town.bankhistory` |
| `/ta town trust <town> <add\|remove> <player>` | Manage trusted players in a town. | `towny.command.townyadmin.town.trust` |

| `/ta town forsale <town> <price>` | Put a town up for sale. | `towny.command.townyadmin.town.forsale` |
| `/ta town notforsale <town>` | Remove a town from sale. | `towny.command.townyadmin.town.notforsale` |

| `/ta town settownlevel <town> <level>` | Override the town level. | `towny.command.townyadmin.town.settownlevel` |
| `/ta town giveboughtblocks <town> <n>` | Give a town bought town blocks. | `towny.command.townyadmin.town.giveboughtblocks` |
| `/ta town checkoutposts <town>` | Validate a specific town's outpost data. | `towny.command.townyadmin.town.checkoutposts` |

```bash
# Create a new town for Steve at his location
/ta town new Riverside Steve

# Force-delete a town
/ta town delete Riverside

# Merge Riverside into Oaktown
/ta town merge Riverside Oaktown

# Move a town's mayor
/ta town set Riverside mayor Steve

# Toggle mobs in a town from the console
/ta town toggle Riverside mobs

# Grant Steve the Assistant rank in Riverside
/ta town rank Riverside add Steve Assistant
```

## Economy administration (`/ta eco`)

| Command | Description | Permission node |
|---|---|---|
| `/ta eco depositall <amount>` | Deposit money into all town and nation banks. | `towny.command.townyadmin.eco.depositall` |
| `/ta eco depositalltowns <amount>` | Deposit money into all town banks. | `towny.command.townyadmin.eco.depositalltowns` |
| `/ta eco depositallnations <amount>` | Deposit money into all nation banks. | `towny.command.townyadmin.eco.depositallnations` |
| `/ta eco resetbanks` | Reset all town and nation bank balances to zero. | `towny.command.townyadmin.eco.resetbanks` |
| `/ta eco convert` | Convert economy data between providers. | `towny.command.townyadmin.eco.convert` |
| `/ta eco info` | Show economy provider information. | `towny.command.townyadmin.eco.info` |

<Warning>
`/ta eco resetbanks` sets every town and nation bank balance to zero. This cannot be undone. Back up your data first.
</Warning>

```bash
# Give every town 500 as a stimulus
/ta eco depositalltowns 500

# Check which economy provider is active
/ta eco info

# Reset all bank balances
/ta eco resetbanks
```

## Maintenance commands

```bash
# Trigger a data backup right now
/ta backup

# Reload all config files without restarting
/ta reload

# Manually fire the new-day event (taxes, upkeep)
/ta newday

# Manually fire the new-hour event
/ta newhour

# Remove residents who have been offline more than 30 days
/ta purge 30

# Check whether Steve has a permission node
/ta checkperm Steve towny.command.town.claim.town

# Give Riverside 5 bonus town blocks
/ta givebonus Riverside 5

# Forcibly unclaim the chunk you are standing on
/ta unclaim

# Teleport to a specific plot by world coordinates
/ta tpplot

# Verify all outpost claim data
/ta checkoutposts
```
