title: "Resident command reference (/res, /resident)"
description: "Full reference for /resident and its aliases /res, /p, and /player — view profiles, manage friends, set modes, toggle plot flags, and handle jail bail."
---

The resident commands let every player interact with their own profile and manage personal settings. Use `/res` as the shortest alias; `/resident`, `/p`, and `/player` are all equivalent.

<Note>
Most resident subcommands require the player to be registered as a Towny resident (i.e. have joined a Towny-enabled world). Looking up another resident requires `towny.command.resident.otherresident`, which is granted to all players by default.
</Note>

## Viewing resident status

```bash
# View your own profile
/res

# View another player's profile
/res Steve
```

The status screen shows your town, nation, registered date, friends, plot count, and active toggle states.

---

## Friend management

Friends receive extra plot permissions when a resident sets their plot to allow friends.

```bash
# Add a friend
/res friend add Steve

# Remove a friend
/res friend remove Steve

# See your full friend list
/res friend list

# Remove all friends at once
/res friend clear
```

## Setting modes

Modes change what information is displayed as you move around the world. Multiple modes can be active simultaneously.

<AccordionGroup>
  <Accordion title="Available modes">

| Mode | Description |
|---|---|
| `map` | Continuously display the ASCII map. |
| `plotborder` | Flash particles on plot borders when entered. |
| `constantplotborder` | Always show plot border particles. |
| `townborder` | Show the outer town perimeter only. |
| `townclaim` | Highlight newly claimed/unclaimed land. |
| `townunclaim` | Highlight unclaimed area on the map. |
| `plotgroup` | Display plot group info on border cross. |
| `district` | Display district info on border cross. |
| `infotool` | Enable left-click info lookup tool. |

  </Accordion>
</AccordionGroup>

```bash
# Enable the map mode
/res set mode map

# Enable multiple modes
/res set mode map plotborder

# Clear all active modes
/res set mode clear

# Reset modes to defaults
/res set mode reset
```

## Toggle flags

Toggles set the default behaviour for resident-owned plots. Individual plots can override these using `/plot toggle`.

```bash
# Prevent explosions on your plots
/res toggle explosion

# Explicitly turn fire spread off
/res toggle fire off

# Re-enable mob spawning
/res toggle mobs on

# Disable PvP on your plots
/res toggle pvp off
```

## Jail bail

If you are jailed in a town that allows bail payments, you can pay your way out.

<Note>
The bail amount is configured per town. You must have sufficient funds in your personal economy account.
</Note>

```bash
/res jail paybail
```
