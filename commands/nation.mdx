Nations sit at the top of Towny's hierarchy, grouping towns together under a King. The `/nation` command (aliases `/n` and `/nat`) controls everything from founding a nation and inviting towns, to managing allies, enemies, taxes, and bank transactions.

<Note>
Most subcommands require you to be a member of a nation. Administrative actions (add, kick, set, toggle) typically require the King or a resident holding the appropriate nation rank. Permission nodes follow the pattern `towny.command.nation.<subcommand>`.
</Note>

## Creating and joining nations

<Warning>
Creating a nation costs money (configured by the admin). Deleting a nation is permanent — all member towns lose their nation membership immediately.
</Warning>

```bash
# Found a new nation named Arcadia
/nation new Arcadia

# Invite a town to join
/nation invite Riverdale

# Accept an incoming ally request (from the ally command)
/nation ally accept OtherNation

# Join an open nation with your town
/nation join Arcadia

# Leave the current nation (as a mayor)
/nation leave
```

---

## Managing member towns

```bash
# Add a town directly (bypasses invite)
/nation add Riverdale

# Remove a town from the nation
/nation kick Riverdale

# Move the capital to a different town
/nation set capital Riverdale

# Transfer kingship to another resident
/nation set king Steve
```

## Diplomacy

<Tip>
Alliance requests must be accepted by both nations before an alliance is active. Enemy declarations are one-sided.
</Tip>

```bash
# Send an ally request
/nation ally add Britannia

# Accept an ally request from Britannia
/nation ally accept Britannia

# Deny an ally request
/nation ally deny Britannia

# Remove an existing alliance
/nation ally remove Britannia

# Declare a nation as an enemy
/nation enemy add Mordor

# Remove the enemy declaration
/nation enemy remove Mordor

# See all current allies
/nation allylist

# See all current enemies
/nation enemylist
```

## Nation bank

```bash
# Deposit 1000 into the nation bank
/nation deposit 1000

# Withdraw 500 from the nation bank (king only)
/nation withdraw 500

# View the last page of transactions
/nation bankhistory

# Show top nations by balance
/nation baltop
```

## Ranks

```bash
# Grant a resident the Chancellor rank
/nation rank add Steve Chancellor

# Remove the rank
/nation rank remove Steve Chancellor

# List all ranked residents
/nation ranklist
```

## Nation settings (`/nation set`)

```bash
# Set the message-of-the-day
/nation set board Join our Discord: discord.gg/example

# Rename the nation
/nation set name NewArcadia

# Set a short tag
/nation set tag ARC

# Set flat daily town tax
/nation set taxes 50

# Set spawn cost
/nation set spawncost 25
```

---

## Nation toggles

```bash
# Toggle neutrality (costs upkeep if enabled)
/nation toggle neutral

# Make the nation spawn publicly accessible
/nation toggle public

# Allow towns to join without an invite
/nation toggle open

# Use percentage-based taxes
/nation toggle taxpercent
```

---

## Listing and information

```bash
# List all nations by residents (default)
/nation list

# Sort by balance
/nation list balance

# Sort by number of towns
/nation list towns

# Other sort options: name, residents, open, public, townblocks, founded, online, upkeep

# View all towns in your nation
/nation townlist

# List all online nation members
/nation online
```

---

## Sanctions

Sanctioning a town restricts certain interactions between the nation and that town.

```bash
# Sanction a town
/nation sanctiontown add RivalTown

# Remove the sanction
/nation sanctiontown remove RivalTown
```
